# Bpm List

| Property                        | Description                     | 
|---------------------------------|---------------------------------|
| ``bpm``                         | Gets all Bpms in the list       |     
| ``metronome``                   | Gets all Metronomes in the list |
| ``to_timing_map()``             | Converts to a ``TimingMap``     |
| ``ave_bpm(last_offset: float)`` | Finds the average BPM.          |
