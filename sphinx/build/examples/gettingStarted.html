Getting Started¶

Installation

If you haven’t installed reamber you can just run

pip install reamber

in your terminal

Requirements¶

If pip doesn’t install extra dependencies for you, you also require these libraries.

  • numpy

    • For common mathematical operations and the numpy.Series

  • pyyaml

    • To handle Quaver files easily

  • pandas

    • To facilitate some algorithms and allow conversion to pandas.DataFrame

  • matplotlib

    • To allow plotting on some algorithms (e.g. npsPlot)

  • pillow

    • To enable drawing on some algorithms (e.g. PlayField)
