Welcome to reamber(Py)’s documentation :)¶

reamber is aiming to be a universal VSRG python library. Here you’ll find all the info that you require to get started in using this library.

New to using the package? Get Started Here

Supported VSRGs¶

There are a few VSRGs this library can work with,

  • osu!

    • Only works with mania

    • Excludes storyboard

  • Quaver

  • StepMania

  • O2Jam

    • Only works with loading OJN files.

  • BMS

    • Does not parse samples/mines/bga
