BMS¶

There are few features excluded intentionally:

  • Sample Loading

  • Easy Sample Writing (That means you can only write notes, you’d have to insert the song/samples manually)

  • BGA

  • Mines

Examples¶

1. Read and Write¶

There are different channel configurations for BMS-style maps. More info in the BMSChannel Page

1 from reamber.quaver.BMSMap import BMSMap
2
3 bms = BMSMap.read_file("path/to/file.bme", noteChannelConfig=BMSChannel.BME)
4 bms.write_file("fileOut.bme")

2. Prints all the BPMs¶

1from reamber.bms.BMSMap import BMSMap
2
3bms = BMSMap.read_file("path/to/file.bme", noteChannelConfig=BMSChannel.BME)
4print(bms.bpm.bpm())

3. Move all columns to the right by 1¶

1from reamber.bms.BMSMap import BMSMap
2
3bms = BMSMap.read_file("path/to/file.bme", noteChannelConfig=BMSChannel.BME)
4
5for hit in bms.notes.hits():
6    hit.column += 1
7for hold in bms.notes.holds():
8    hold.column += 1
