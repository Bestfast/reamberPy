Scroll Speed¶

Gets the scroll speed and visualizes it on matplotlib

from reamber.osu.OsuMap import OsuMap
import matplotlib.pyplot as plt
import pandas as pd

m = OsuMap()
m.readFile("path/to/file.osu")
pd.DataFrame(m.scrollSpeed()).set_index('offset').plot()
plt.show()

This function gets the resulting scroll speed from the map as a dictionary

[{'offset': OFFSET, 'speed': SPEED}, {'offset': OFFSET, 'speed': SPEED}, ... ]

Module Info¶

class reamber.base.Bpm.Bpm(offset: float = 0.0, bpm: float = 120.0, metronome: int = 4)¶

Bases: reamber.base.Timed.Timed

A non-playable timed object that specifies the tempo of the map.

This is synonymous with Bpm Point, it’s named Object to make it consistent

static alignBpms(bpms: List[reamber.base.Bpm.Bpm], BEAT_ERROR_THRESHOLD: float = 0.001, BEAT_CORRECTION_FACTOR: float = 5.0) → List[reamber.base.Bpm.Bpm]¶

Ensures that all BPMs are on an integer measure by adding or amending

Parameters

  • bpms – The BPMs

  • BEAT_ERROR_THRESHOLD – If the fraction change in beat’s is more than this, we append a new BPM, else we create a new bpm. i.e. if there’s too many high BPM points, then increase this

  • BEAT_CORRECTION_FACTOR – The number of beats to search prior to the affected beat to amend. i.e. if there are too many incorrectly snapped notes DUE TO ADDED BPMs then decrease this

Returns

The new and old BPMs
