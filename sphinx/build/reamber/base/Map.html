Activity¶

This algorithm finds out how long each Bpm is active and returns a List of tuple of (Bpm, ActiveTimeMs).

The last Bpm is calculated by the last Note present if not overridden.

Input

1
2
3
4
5
from reamber.osu.OsuMap import OsuMap

osu = OsuMap()
osu.readFile("path/to/file.osu")
print(osu.activity())

Module Info¶

class reamber.base.Map.Map¶

Bases: abc.ABC

This class should be inherited by all Map Objects

They must inherit the data method, which extracts all data they hold. They are also assumed to be a TimedList.

activity(lastOffset: float = None) → List[Tuple[Bpm, float]]¶

Calculates how long the Bpm is active. Implicitly sorts BPM

For example

The algorithm calculates this:

SEC 1   2   3   4   5   6   7   8   9
BPM 100 ------> 200 --> 300 -------->

returns [(BPMPoint<100>, 3000), (BPMPoint<200>, 2000), (BPMPoint<300>, 3000)]

Parameters

lastOffset – If not None, then this offset will be used to terminate activity, else last offset will be used.

Returns

A List of Tuples in the format [(BPMPoint, Activity In ms), …]
