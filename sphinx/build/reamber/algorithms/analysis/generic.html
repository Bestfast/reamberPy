from reamber.osu.OsuMap import OsuMap
import matplotlib.pyplot as plt

from reamber.algorithms.analysis.generic.rollingDensity import rollingDensity
import os

m = OsuMap()
m.readFile("path/to/file.osu")

rollingDensity(m.notes.hits().offsets(), rollingWindowS=2).plot()
plt.show()

Module Info¶

reamber.algorithms.analysis.generic.rollingDensity.rollingDensity(offsets: List[float], rollingWindowS: float = None) → pandas.core.series.Series¶

Returns the Density Series for any list

Parameters

  • offsets – Any List of offsets

  • rollingWindowS – The window to search in seconds. If left as None, the window is 0

Returns
