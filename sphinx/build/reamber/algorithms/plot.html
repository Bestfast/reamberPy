Plotting¶

Most plotting functions are designed to not show the plot directly, you’d need to invoke matplotlib.pyplot.show().

Notes Per Second (NPS) Plotting¶

Example¶

Input

1 from reamber.algorithms.plot.nps import nps_plot
2
3 from reamber.osu.OsuMap import OsuMap
4 import matplotlib.pyplot as plt
5
6 m = OsuMap.read_file("path/to/file.osu")
7 nps_plot(m, binSize=500)
8 plt.show()

Output

Module Info¶

reamber.algorithms.plot.nps.nps_plot(pkg, ax: Optional[matplotlib.axes._axes.Axes] = None, window=1000, stride=None, legend=True, tick_step_size=60000, bar_kwargs=None) → matplotlib.axes._axes.Axes¶

This creates an NPS Plot with the axes.

Parameters

  • pkg – Any Note Package

  • ax – The axis to plot on, if None, we use gca()

  • window – The window of the roll

  • stride – The stride length of the roll

  • legend – Whether to show the legend

  • tick_step_size – How many milliseconds per tick

  • bar_kwargs – The kwargs to pass into plot()

reamber.algorithms.plot.nps.nps_plot_by_key(pkg, fig: Optional[matplotlib.figure.Figure] = None, shape: Optional[Tuple] = None, window=1000, stride=None, title=True, legend=True, bar_kwargs=None) → matplotlib.figure.Figure¶

This creates an NPS Plot with the axes.

Parameters

  • pkg – Any Note Package

  • fig – The figure to plot on, if None, we use gcf()

  • shape – The shape of the axes to take. (rows, columns)

  • window – The window of the roll

  • stride – The stride length of the roll

  • title – Whether to show the key titles

  • legend – Whether to show legend. False to show none, True to show on first, ‘all’ to show on all

  • bar_kwargs – The kwargs to pass into plot()
