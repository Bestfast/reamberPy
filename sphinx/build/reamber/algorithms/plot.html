Plotting¶

Most plotting functions are designed to not show the plot directly, you’d need to invoke matplotlib.pyplot.show().

Notes Per Second (NPS) Plotting¶

Example¶

Input

1
2
3
4
5
6
7
8
 from reamber.algorithms.plot.nps import npsPlot

 from reamber.osu.OsuMap import OsuMap
 import matplotlib.pyplot as plt

 m = OsuMap.readFile("path/to/file.osu")
 npsPlot(m, binSize=500)
 plt.show()

Module Info¶

reamber.algorithms.plot.nps.npsPlot(pkg: reamber.base.lists.NotePkg.NotePkg, ax: matplotlib.axes._axes.Axes = None, window=1000, stride=None, legend=True, tickStepSize=60000, barKwargs=None) → matplotlib.axes._axes.Axes¶

This creates an NPS Plot with the axes.

Parameters

  • pkg – Any Note Package

  • ax – The axis to plot on, if None, we use gca()

  • window – The window of the roll

  • stride – The stride length of the roll

  • legend – Whether to show the legend

  • tickStepSize – How many milliseconds per tick

  • barKwargs – The kwargs to pass into plot()

reamber.algorithms.plot.nps.npsPlotByKey(pkg: reamber.base.lists.NotePkg.NotePkg, fig: matplotlib.figure.Figure = None, shape: Tuple = None, window=1000, stride=None, title=True, legend=True, barKwargs=None) → matplotlib.figure.Figure¶

This creates an NPS Plot with the axes.

Parameters

  • pkg – Any Note Package

  • fig – The figure to plot on, if None, we use gcf()

  • shape – The shape of the axes to take. (rows, columns)

  • window – The window of the roll

  • stride – The stride length of the roll

  • title – Whether to show the key titles

  • legend – Whether to show legend. False to show none, True to show on first, ‘all’ to show on all

  • barKwargs – The kwargs to pass into plot()
