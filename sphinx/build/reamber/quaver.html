3. Multiply all Bpms and Svs by 1.5¶

from reamber.quaver.QuaMap import QuaMap

q = QuaMap.readFile("file.qua")

for sv in q.svs.data():
    sv.multiplier *= 1.5

for bpm in q.bpms.data():
    bpm.bpm *= 1.5

metadata(unicode=True) → str¶

Grabs the map metadata

Parameters

unicode – Whether to try to find the unicode or non-unicode. This doesn’t try to convert unicode to ascii, it just looks for if there’s an available translation.

Returns

rate(by: float, inplace: bool = False)¶

Changes the rate of the map

Parameters

  • by – The value to rate it by. 1.1x speeds up the song by 10%. Hence 10/11 of the length.

  • inplace – Whether to perform the operation in place. Returns a copy if False

static readFile(filePath: str) → reamber.quaver.QuaMap.QuaMap¶

Reads a .qua, loads inplace, hence it doesn’t return anything

Parameters

scrollSpeed(centerBpm: Optional[float] = None) → List[Dict[str, float]]¶

Evaluates the scroll speed based on mapType. Overrides the base to include SV

e.g. if BPM == 200.0 and CenterBPM == 100.0, it’ll return {‘offset’: X, ‘speed’: 2.0}

Parameters

centerBpm – The bpm to zero calculations on. If None, it’ll just be the multiplication of bpm and sv.

Returns

Returns a list dict of keys offset and speed
