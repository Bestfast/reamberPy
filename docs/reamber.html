static alignBpms(bpms: List[reamber.base.BpmObj.BpmObj], BEAT_ERROR_THRESHOLD: float = 0.001, BEAT_CORRECTION_FACTOR: float = 5.0) → List[reamber.base.BpmObj.BpmObj]¶

Ensures that all BPMs are on an integer measure by adding or amending

Parameters

  • bpms – The BPMs

  • BEAT_ERROR_THRESHOLD – If the fraction change in beat’s is more than this, we append a new BPM, else we create a new bpm. i.e. if there’s too many high BPM points, then increase this

  • BEAT_CORRECTION_FACTOR – The number of beats to search prior to the affected beat to amend. i.e. if there are too many incorrectly snapped notes DUE TO ADDED BPMs then decrease this

Returns

The new and old BPMs
