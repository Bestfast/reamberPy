Rate Changer¶

Example¶

m = OsuMapObj()
m.readFile("../rsc/maps/osu/Caravan.osu")
rate(m, 2.0, inplace=True)
m.writeFile("Caravan200.osu")

Can’t find a better name for this without it being too vague or generic

reamber.algorithms.mutate.rate.rate(m: OsuMapObj, by: float, inplace: bool = 'False') → OsuMapObj¶
reamber.algorithms.mutate.rate.rate(m: QuaMapObj, by: float, inplace: bool = 'False') → QuaMapObj
reamber.algorithms.mutate.rate.rate(s: SMMapSetObj, by: float, inplace: bool = 'False') → SMMapSetObj

Speeds up the map by rate specified.

Parameters

  • m – The map

  • by – The by, 0.8 means 80% speed, 1.2 means 120% speed

  • inplace – If True, modifies the map in place without returning
