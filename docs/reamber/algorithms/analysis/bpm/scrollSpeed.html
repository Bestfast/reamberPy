Scroll Speed¶

reamber.algorithms.analysis.bpm.scrollSpeed.scrollSpeed(m: OsuMapObj, centerBpm: float = 'None') → List[Dict[str, float]]¶
reamber.algorithms.analysis.bpm.scrollSpeed.scrollSpeed(m: SMMapObj, centerBpm: float = 'None') → List[Dict[str, float]]
reamber.algorithms.analysis.bpm.scrollSpeed.scrollSpeed(m: O2JMapObj, centerBpm: float = 'None') → List[Dict[str, float]]
reamber.algorithms.analysis.bpm.scrollSpeed.scrollSpeed(m: QuaMapObj, centerBpm: float = 'None') → List[Dict[str, float]]

Evaluates the scroll speed based on mapType

e.g. if BPM == 200.0 and CenterBPM == 100.0, it’ll return {‘offset’: X, ‘speed’: 2.0}

Parameters

  • m – The Map Object

  • centerBpm – The bpm to zero calculations on. If None, it’ll just be the multiplication of bpm and sv.

Returns

Returns a list dict of keys offset and speed

 from reamber.osu.OsuMapObj import OsuMapObj
 from reamber.algorithms.analysis.bpm.scrollSpeed import scrollSpeed
 import matplotlib.pyplot as plt
 import pandas as pd

 def test_osu(self):
     m = OsuMapObj()
     m.readFile("file.osu")
     pd.DataFrame(scrollSpeed(m)).set_index('offset').plot()
     plt.show()

This function gets the resulting scroll speed from the map as a dictionary
