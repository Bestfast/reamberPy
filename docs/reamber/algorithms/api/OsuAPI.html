Osu API¶

There are 2 versions, currently, V1 is more supported because V2 is very volatile.

All necessary information about the API can be found here. osu! API Wiki

Example¶

How to set up the ini/cfg file is detailed in API

from reamber.algorithms.api.osu.OsuAPIV1 import OsuAPIV1
import json

api = OsuAPIV1.fromCfg(cfg_path="api.ini")

req = api.getBeatmaps(m=3, since="2020-08-01")
content = json.loads(req.content)
print(content)

Gets beatmaps.

Parameters

  • since – return all beatmaps ranked or loved since this date. Must be a MySQL date. In UTC

  • s – Beatmapset_id

  • b – Beatmap_id

  • u – User_id or a username

  • type – specify if u is a user_id or a username. Use string for usernames or id for user_ids.

  • m – Mode (0 = osu!, 1 = Taiko, 2 = CtB, 3 = osu!mania).

  • a – Include converted beatmaps (0 = not included, 1 = included).

  • h – The beatmap hash.

  • limit – Amount of results. Default and maximum are 500.

  • mods – Beatmap Mods

static fromCfg(cfg_path)¶

This grabs the token from the current cfg, good if you don’t want to keep on requesting tokens

However note that the token will expire in 24 hours upon request.
