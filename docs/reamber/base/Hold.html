Hold¶

class reamber.base.Hold.Hold(offset: float, column: int, length: float, **kwargs)¶

Bases: reamber.base.Note.Note

A held timed object with a length.

Notes:

We only store the length, the tail offset is calculated. We inherit Note instead of Hit

Examples:

>>> h = Hold(offset=1000, column=1, length=1000)
>>> h.tail_offset
2000
__init__(offset: float, column: int, length: float, **kwargs)¶

Initializer

Args:

offset: Offset in ms column: Column length: Length in ms

property tail_offset: float¶

Offset of the tail in ms

Notes:

This is simply offset + length

float of Tail offset in ms
