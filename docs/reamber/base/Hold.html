Hold¶

class reamber.base.Hold.Hold(offset: float, column: int, length: float, **kwargs)¶

Bases: reamber.base.Note.Note

A holdable timed object with a specified length.

We only store the length, the tail offset is calculated.

We don’t directly inherit Hit because the inheritance may be confusing, we’ll just subclass Note.

class reamber.base.Hold.HoldTail(offset: float, column: int, length: float, **kwargs)¶

Bases: reamber.base.Note.Note

The purpose of this class is to be able to detect the tail as a separate object instead of just Hold

This class however, is entirely disconnected from Hold, and should only be used for convenience as using the head is more natural.
