activity(last_offset: Optional[float] = None) → numpy.ndarray¶

Calculates how long each Timed Object is active.

For example:

The algorithm calculates this:

SEC 1   2   3   4   5   6   7   8   9
BPM 100 ------> 200 --> 300 -------->
Parameters

last_offset – Last offset, if None, uses Timed.last_offset()

Returns

np.ndarray of the activity/offset differences.

rolling_density(window: float, stride: Optional[float] = None, first_offset: Optional[float] = None, last_offset: Optional[float] = None) → Dict[int, int]¶

Returns the Density Dictionary

Parameters

  • window – The window to search in milliseconds.

  • stride – The stride length of each search in milliseconds, if None, stride = window

  • first_offset – The first offset to start search on. If None, first_offset will be used.

  • last_offset – The last offset to end search on. If None, last_offset will be used. (The search will intentionally exceed if it doesn’t fit.)

Returns

Dictionary of offset as key and count as value
