rollingDensity(window: int = 1000, stride: Optional[int] = None, firstOffset: Optional[float] = None, lastOffset: Optional[float] = None) → Dict[str, Dict[int, int]]¶

Returns the Density List Dictionary

First offset and last offset is recalculated here in Package to make sure that the indexes are consistent.

Parameters

  • window – The window to search in milliseconds.

  • stride – The stride length of each search in milliseconds, if None, stride = window

  • firstOffset – The first offset to start search on. If None, firstOffset will be used.

  • lastOffset – The last offset to end search on. If None, lastOffset will be used. (The search will intentionally exceed if it doesn’t fit.)

Returns

Dictionary of offset as key and count as value
